1. Purpose

The purpose of this model is to represent "authorisations" by which is meant:

a Rule that grants Actions to an assignee Agent over an Authorised Asset

This model is just a specialisation of the Open Data Rights Language (ODRL) model, but it was though necessary to make this model, rather than to just point to ODRL for use, for while ODRL work perfectly well for the scenarios exemplified here (see BAsic Concepts for expected use below), ODRL's own examples are all oriented elsewhere - to the use of data objects.

This model aims to use names for elements and descriptions of them that are more intuitive within its target domains.

2. Basic Concepts

Typical use is expected to be modelling of permissions granting to people and companies for activities such as mining prospecting and the modelling of powers of government.

The things granting the authorisation are Authorities: laws, policies, regulations etc.

Authorisation c

IRI https://linked.data.gov.au/def/autho/Authorisation
Description

An Authorisation is a Rule that grants Actions to an assignee Agent over an Authorised Asset
Scope Note

Use thi class to indicate the actions that the Authorisation permits the Assignee
Example 
  •                 # A Geographical Name is named by a Minister,
                # in accordance with a Place Names Act

                PREFIX : <https://linked.data.gov.au/def/autho/>
                PREFIX ex: <http://example.com/>
                PREFIX gn: <https://linked.data.gov.au/def/gn/>
                PREFIX odrl: <http://www.w3.org/ns/odrl/2/>
                PREFIX rdfs: <http://www.w3.org/2000/01/rdf-schema#>

                gn:GeographicalName
                    rdfs:subclassOf :AuthorisedAsset ;
                .

                ex:kgari
                    a gn:GeographicalName ;
                    :hasAuthorisation ex:placeNamesAct1994 ;
                .

                ex:placeNamesAct1994
                    a :Authority ;
                    odrl:permission ex:naming ;
                .

                ex:naming
                    a :Authorisation ;
                    odrl:action ex:giveANameToAPlace ;
                    odrl:assignee :ministerForResources ;
                .

                ex:giveANameToAPlace
                    a odrl:Action ;
                .

                :ministerForResources
                    a odrl:Party ;
                .
            
  •                 # A Mining Permit allows a Miner to prospect
                # a given Permit Area, but the Miner must
                # pay for any damage

                PREFIX : <https://linked.data.gov.au/def/autho/>
                PREFIX ex: <http://example.com/>
                PREFIX gn: <https://linked.data.gov.au/def/gn/>
                PREFIX mautho: <https://linked.data.gov.au/def/mining-autho/>
                PREFIX odrl: <http://www.w3.org/ns/odrl/2/>
                PREFIX rdfs: <http://www.w3.org/2000/01/rdf-schema#>

                ex:permitAreaX
                    a mautho:PermitArea ;
                    :hasAuthorisation ex:miningAct178 ;
                .

                ex:miningAct178
                    a :Authority ;
                    odrl:permission ex:prospectingRight ;
                    odrl:obligation ex:prospectingDuty ;
                .

                ex:prospectingRight
                    a :Authorisation ;
                    odrl:action
                        ex:prospectForMinerals ,
                        ex:conductGeologicalMapping ;
                    odrl:assignee :minerY ;
                    odrl:duty ex:prospectingDuty ;
                .

                ex:prospectForMinerals
                    a odrl:Action ;
                .

                ex:conductGeologicalMapping ;
                    a odrl:Action ;
                .

                :minerY
                    a odrl:Party ;
                .

                ex:prospectingDuty
                    a odrl:Duty ;
                    odrl:action ex:payForDamages ;
                .

                ex:payForDamages
                    a odrl:Action ;
                .
